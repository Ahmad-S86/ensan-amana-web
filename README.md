# Al Ensan Amana Association Website

This is the source code for the Al Ensan Amana Association website.

## Structure

- **index.html**: Main HTML file containing all the website sections.
- **styles.css**: CSS file for styling the website with a professional, NGO-focused design system.
- **script.js**: JavaScript file for interactive elements (mobile menu, animations).

## How to Run

1.  Simply open `index.html` in any modern web browser.
2.  For a better development experience, use a local server (e.g., Live Server extension in VS Code).

## Design Choices

-   **Colors**: Deep Teal/Green for trust and growth, with a warm accent color (Gold/Terracotta).
-   **Typography**: Inter (Google Fonts) for clean, modern readability.
-   **Imagery**:
    -   Placeholders are used for backgrounds.
    -   **Logo**: Save the official logo as `logo.png` in the detailed directory to display it in the header.
